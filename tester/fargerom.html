OKLCH vs HSL

Betegnelser

OKLCH
It is called the Oklab color space, because it is an OK Lab color space.
OKLCH står for OK, lightness, chroma, hue.
Introdusert i CSS Color Module Level 4
HSL
Står for hue, saturation lightness
